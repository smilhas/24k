## Steps for executing app localy
# (emulating Azure Static Web App)

    1. cd client
    2. npm run build
    3. cd ..
    4. swa start client/build --api functions


# Just web app

    1. cd client
    2. npm start


# Just functions

    1. cd fucntions
    2. func start
